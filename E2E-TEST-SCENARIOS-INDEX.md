# 📋 메모 앱 E2E 테스트 시나리오 인덱스

`http://localhost:3000` 메모 앱을 위한 종합적인 E2E 테스트 시나리오 모음입니다.

## 🎯 테스트 시나리오 목록

### 1. 핵심 기능 테스트

#### 📝 [메모 생성 기능](./memo-create-scenario.md)
- **목적**: 새로운 메모를 성공적으로 작성하고 저장할 수 있는지 확인
- **주요 테스트**: 폼 작성, 카테고리 선택, 태그 추가, 마크다운 입력, 저장 프로세스

#### 🔍 [메모 검색 기능](./memo-search-scenario.md)
- **목적**: 검색어를 입력했을 때 해당하는 메모들이 올바르게 필터링되는지 확인
- **주요 테스트**: 실시간 검색, 검색 결과 없음 케이스, 검색 초기화

#### 🏷️ [카테고리 필터 기능](./memo-category-filter-scenario.md)
- **목적**: 카테고리별로 메모를 필터링하는 기능이 올바르게 작동하는지 확인
- **주요 테스트**: 카테고리별 필터링, 통계 정보, 빈 카테고리 처리

#### 👁️ [메모 상세 보기 모달](./memo-view-modal-scenario.md)
- **목적**: 메모 카드를 클릭했을 때 상세 내용을 모달로 볼 수 있는지 확인
- **주요 테스트**: 모달 열기/닫기, 내용 표시, 키보드/마우스 인터랙션

#### ✏️ [메모 편집 기능](./memo-edit-scenario.md)
- **목적**: 기존 메모를 편집하고 수정 사항이 올바르게 저장되는지 확인
- **주요 테스트**: 데이터 로드, 내용 수정, 저장, 취소 기능

#### 🗑️ [메모 삭제 기능](./memo-delete-scenario.md)
- **목적**: 메모를 안전하게 삭제할 수 있고 확인 절차가 올바르게 작동하는지 확인
- **주요 테스트**: 삭제 확인 다이얼로그, 삭제 취소, 통계 업데이트

### 2. 고급 기능 테스트

#### ✅ [폼 유효성 검사](./memo-validation-scenario.md)
- **목적**: 메모 작성 시 필수 항목들이 올바르게 검증되는지 확인
- **주요 테스트**: 빈 필드 검증, 공백 처리, 중복 태그 방지

#### 📄 [마크다운 에디터 기능](./memo-markdown-editor-scenario.md)
- **목적**: 마크다운 에디터가 올바르게 작동하고 콘텐츠가 정상적으로 렌더링되는지 확인
- **주요 테스트**: 마크다운 문법, HTML 렌더링, 복잡한 구조 처리

#### 📱 [반응형 UI 및 사용성](./memo-responsive-ui-scenario.md)
- **목적**: 애플리케이션이 다양한 화면 크기에서 올바르게 작동하는지 확인
- **주요 테스트**: 데스크톱/태블릿/모바일 뷰, 터치 인터랙션, 레이아웃 적응

### 3. 통합 테스트

#### 🔄 [통합 사용자 워크플로우](./memo-integration-scenario.md)
- **목적**: 실제 사용자가 수행할 만한 전체적인 워크플로우를 통해 모든 기능이 함께 잘 작동하는지 확인
- **주요 테스트**: 전체 CRUD 워크플로우, 기능 간 연동, 데이터 지속성

## 🔧 테스트 실행 가이드

### 전제 조건
- 메모 앱이 `http://localhost:3000`에서 실행 중이어야 함
- Supabase 백엔드가 정상적으로 연결되어 있어야 함

### 권장 테스트 순서

1. **기본 기능 검증**
   - 메모 생성 → 상세 보기 → 편집 → 삭제 순서로 테스트

2. **검색 및 필터 기능**
   - 메모가 충분히 생성된 후 검색/필터 테스트 수행

3. **고급 기능 및 에지 케이스**
   - 유효성 검사, 마크다운, 반응형 UI 테스트

4. **통합 테스트**
   - 모든 개별 기능 테스트 완료 후 전체 워크플로우 테스트

### 🐛 테스트 실패 시 확인사항

- 브라우저 콘솔 에러 확인
- 네트워크 탭에서 API 호출 상태 확인
- Supabase 연결 상태 확인
- 로컬 서버 상태 확인

## 📊 메모 앱 주요 기능 요약

- **메모 관리**: 생성, 읽기, 수정, 삭제 (CRUD)
- **카테고리 시스템**: 개인, 업무, 학습, 아이디어, 기타
- **검색 기능**: 제목, 내용, 태그 기반 실시간 검색
- **마크다운 지원**: 풀 기능의 마크다운 에디터와 렌더링
- **태그 시스템**: 메모별 태그 추가/제거
- **반응형 디자인**: 데스크톱/태블릿/모바일 지원
- **통계 정보**: 카테고리별 메모 개수, 필터링 결과 수

이 시나리오들을 통해 메모 앱의 모든 핵심 기능과 사용자 경험을 체계적으로 검증할 수 있습니다.
