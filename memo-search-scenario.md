# E2E 테스트 시나리오: 메모 검색 기능

## 목적

검색어를 입력했을 때 해당하는 메모들이 올바르게 필터링되는지 확인

## 시나리오 단계

### 검색 기능 테스트

1. 브라우저에서 http://localhost:3000 접속
2. 메모 목록이 로드될 때까지 대기
3. 검색 입력란에 "테스트" 키워드 입력
4. 검색 결과가 실시간으로 필터링되는지 확인
5. 검색 결과 통계 정보 확인 ("X개 메모 (전체 Y개 중)")
6. 검색된 메모들의 제목이나 내용에 "테스트"가 포함되어 있는지 확인

### 검색 결과 없음 케이스

7. 검색 입력란에 존재하지 않는 키워드 (예: "존재하지않는키워드") 입력
8. "검색 결과가 없습니다" 메시지 표시 확인
9. "다른 검색어나 카테고리를 시도해보세요." 안내 메시지 확인

### 검색 초기화

10. "필터 초기화" 버튼 클릭
11. 검색어가 지워지고 전체 메모 목록이 다시 표시되는지 확인
12. 통계 정보가 원래대로 복원되는지 확인

## 예상 결과

- 실시간 검색 필터링이 정상 작동
- 검색 결과 통계가 정확히 표시됨
- 검색 결과가 없을 때 적절한 안내 메시지 표시
- 필터 초기화 기능이 정상 작동
