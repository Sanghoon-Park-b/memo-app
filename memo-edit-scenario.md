# E2E 테스트 시나리오: 메모 편집 기능

## 목적

기존 메모를 편집하고 수정 사항이 올바르게 저장되는지 확인

## 시나리오 단계

### 메모 목록에서 편집

1. 브라우저에서 http://localhost:3000 접속
2. 메모 목록이 로드될 때까지 대기
3. 첫 번째 메모 카드의 편집 버튼(연필 아이콘) 클릭
4. 메모 편집 모달이 열리는지 확인
5. 모달 제목이 "메모 편집"으로 표시되는지 확인

### 기존 데이터 확인

6. 제목 입력란에 기존 제목이 표시되는지 확인
7. 카테고리 드롭다운에 기존 카테고리가 선택되어 있는지 확인
8. 마크다운 에디터에 기존 내용이 표시되는지 확인
9. 기존 태그들이 표시되는지 확인

### 내용 수정

10. 제목을 "수정된 메모 제목"으로 변경
11. 카테고리를 다른 카테고리로 변경
12. 마크다운 에디터에서 내용 일부 수정
13. 새로운 태그 "수정됨" 추가
14. 기존 태그 중 하나 삭제

### 저장 및 확인

15. "수정하기" 버튼 클릭
16. 모달이 닫히는지 확인
17. 메모 목록에서 수정된 내용이 반영되었는지 확인
18. 수정된 제목이 표시되는지 확인
19. 변경된 카테고리 배지가 표시되는지 확인
20. 새로운 태그가 표시되는지 확인

### 상세 모달에서 편집

21. 수정된 메모 카드 클릭하여 상세 모달 열기
22. 편집 버튼 클릭
23. 편집 모달이 열리는지 확인
24. 내용 일부 추가 수정
25. "수정하기" 버튼 클릭
26. 상세 모달로 돌아가서 수정된 내용 확인

### 편집 취소

27. 다른 메모의 편집 버튼 클릭
28. 일부 내용 수정
29. "취소" 버튼 클릭
30. 모달이 닫히고 수정사항이 반영되지 않았는지 확인

## 예상 결과

- 기존 메모 데이터가 편집 폼에 올바르게 로드됨
- 수정된 내용이 정확히 저장됨
- 메모 목록과 상세 보기에서 수정사항이 즉시 반영됨
- 취소 기능이 정상 작동하여 변경사항이 저장되지 않음
