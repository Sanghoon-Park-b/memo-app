# E2E 테스트 시나리오: 카테고리 필터 기능

## 목적

카테고리별로 메모를 필터링하는 기능이 올바르게 작동하는지 확인

## 시나리오 단계

### 카테고리별 필터링

1. 브라우저에서 http://localhost:3000 접속
2. 메모 목록이 로드될 때까지 대기
3. 카테고리 드롭다운 메뉴 확인 ("전체 카테고리" 기본 선택)
4. 각 카테고리 옵션에 해당 메모 개수가 표시되는지 확인
5. "개인" 카테고리 선택
6. 개인 카테고리 메모들만 표시되는지 확인
7. 필터링된 메모들의 카테고리 배지가 모두 "개인"인지 확인
8. 통계 정보 업데이트 확인

### 다른 카테고리 테스트

9. "업무" 카테고리 선택
10. 업무 카테고리 메모들만 표시되는지 확인
11. "학습" 카테고리 선택
12. 학습 카테고리 메모들만 표시되는지 확인

### 빈 카테고리 테스트

13. 메모가 없는 카테고리 선택 (있는 경우)
14. "검색 결과가 없습니다" 메시지 표시 확인
15. 적절한 안내 메시지가 표시되는지 확인

### 필터 초기화

16. "필터 초기화" 버튼 클릭
17. 카테고리가 "전체 카테고리"로 리셋되는지 확인
18. 전체 메모 목록이 다시 표시되는지 확인

## 예상 결과

- 선택한 카테고리의 메모들만 정확히 필터링됨
- 카테고리별 메모 개수가 올바르게 표시됨
- 빈 카테고리에 대한 적절한 안내 메시지
- 필터 초기화 기능이 정상 작동
