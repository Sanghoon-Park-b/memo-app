# E2E 테스트 시나리오: 메모 생성 기능

## 목적

새로운 메모를 성공적으로 작성하고 저장할 수 있는지 확인

## 시나리오 단계

1. 브라우저에서 http://localhost:3000 접속
2. 페이지 로딩 완료 대기
3. 우상단 "새 메모" 버튼 클릭
4. 메모 작성 모달이 열리는지 확인
5. 제목 입력란에 "테스트 메모 제목" 입력
6. 카테고리 드롭다운에서 "개인" 선택
7. 마크다운 에디터에 "# 테스트 내용\n\n이것은 **테스트** 메모입니다." 입력
8. 태그 입력란에 "테스트" 입력 후 Enter 키 또는 "추가" 버튼 클릭
9. 태그가 추가된 것을 확인
10. "저장하기" 버튼 클릭
11. 모달이 닫히고 메모 목록에 새 메모가 표시되는지 확인
12. 생성된 메모의 제목, 카테고리, 태그가 올바르게 표시되는지 확인

## 예상 결과

- 모든 입력 데이터가 올바르게 저장됨
- 메모 목록에 새 메모가 즉시 표시됨
- 메모 카드에 제목, 카테고리, 태그, 생성 시간이 표시됨
