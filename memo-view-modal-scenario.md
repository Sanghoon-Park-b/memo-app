# E2E 테스트 시나리오: 메모 상세 보기 모달

## 목적

메모 카드를 클릭했을 때 상세 내용을 모달로 볼 수 있는지 확인

## 시나리오 단계

### 모달 열기

1. 브라우저에서 http://localhost:3000 접속
2. 메모 목록이 로드될 때까지 대기
3. 첫 번째 메모 카드 클릭
4. 메모 상세 보기 모달이 열리는지 확인
5. 모달 배경이 어두워지는지 확인

### 모달 내용 확인

6. 메모 제목이 올바르게 표시되는지 확인
7. 카테고리 배지가 올바른 색상과 텍스트로 표시되는지 확인
8. 생성일/수정일이 올바른 형식으로 표시되는지 확인
9. 메모 내용이 마크다운으로 렌더링되어 표시되는지 확인
10. 태그들이 올바르게 표시되는지 확인 (태그가 있는 경우)

### 모달 액션 버튼

11. 편집 버튼(연필 아이콘)이 표시되는지 확인
12. 삭제 버튼(휴지통 아이콘)이 표시되는지 확인
13. 닫기 버튼(X 아이콘)이 표시되는지 확인

### 모달 닫기 - 닫기 버튼

14. 닫기 버튼 클릭
15. 모달이 닫히고 메인 페이지로 돌아가는지 확인

### 모달 닫기 - 배경 클릭

16. 다른 메모 카드 클릭하여 모달 다시 열기
17. 모달 배경(검은 영역) 클릭
18. 모달이 닫히는지 확인

### 모달 닫기 - ESC 키

19. 다른 메모 카드 클릭하여 모달 다시 열기
20. ESC 키 누르기
21. 모달이 닫히는지 확인

### 스크롤 방지 확인

22. 메모 카드 클릭하여 모달 열기
23. 모달이 열린 상태에서 스크롤 시도
24. 배경 페이지 스크롤이 방지되는지 확인
25. 모달 닫기 후 스크롤이 다시 가능한지 확인

## 예상 결과

- 메모 상세 내용이 올바르게 표시됨
- 마크다운 콘텐츠가 정상 렌더링됨
- 모든 닫기 방법(버튼, 배경, ESC)이 정상 작동
- 모달 열림/닫힘 시 스크롤 제어가 올바르게 작동
